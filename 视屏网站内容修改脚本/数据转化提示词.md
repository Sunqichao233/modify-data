# 视屏网站内容修改脚本：数据转化提示词（覆盖版）

请根据表格内容**重新生成并覆盖** `modify_dataset.csv`，固定输出列：

`filter_field,filter_value,modify_type,modify_content,upload_status`

## 规则

1. `filter_field` 永远写 `title`。  
2. `filter_value` 取右侧每一行标题文本（原样保留）。  
3. `modify_type` 由我指定（例如：`image` / `video`）。  
4. 当 `modify_type=image` 时：  
左侧分类文字最后的数字作为图片编号，`modify_content=<编号>.png`。  
例如左侧是 `... 2`，则 `modify_content=2.png`。  
5. 当 `modify_type=video` 时：  
`modify_content` 由我单独提供（不要自行推断）。  
6. 同一个左侧分类块下的所有右侧行，共用同一个图片编号。  
7. `upload_status` 在生成时默认留空（`""`）。  
8. 输出为标准 CSV，第一行必须是表头，每个右侧条目输出一行。  
9. 默认行为是“覆盖整个文件”，不是追加；除非我明确说“追加”。

## 示例（image）

左侧：`Web3とブロックチェーンの基礎 1`  
右侧：
- `Web3イントロダクション（1）`
- `Web3イントロダクション（2）`

若指定 `modify_type=image`，输出：

```csv
filter_field,filter_value,modify_type,modify_content,upload_status
title,Web3イントロダクション（1）,image,1.png,
title,Web3イントロダクション（2）,image,1.png,
```
